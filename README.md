js语法总结
        1.js的基础数据类型：undefined,null,string,number,boolean
        2.js引用类型：object,function
    js数组方法
        1.push:像数组的末尾添加元素，会改变原数组，返回改变后原数组（新的长度），
        2.unshift:想数组的开头添加元素,会改变原数组，返回改变后的原数组（新长度），
        3.pop:删除数组的最后一位，返回改变后的原数组（新的长度），
        4.shift:删除数组的第一位，返回改变后的原数组（新的长度），
        5.concat:合并两个数组，返回新的数组，不糊改变原来的数组，
        6.join：将数组转换成字符串,不会改变原数组，括号里面的符号可以自己定
        7.reserve:将数组倒序排列,会改变原数组
        8.fill:替换数组中的某个元素，会改变原数组，
        9.includes:用来判断当前数组是否包含某指定的值，如果是，则返回 true，否则返回 false。
        10.slice:会浅复制（shallow copy）数组的一部分到一个新的数组，并返回这个新数组。不会改变原数组
        11.splice:删除或者添加一个元素，会改变原来的数组
        11.indexOf:返回给定元素，能在数组中找到的第一个索引值，否则返回-1。
     string字符串方法
     	1.charAt:返回字符在字符串中的索引
     	2.charCodeAt:返回给定索引对应字符串的Unicode值，
     	3.split:将字符串转换成数组，括号里的字符必须是字符串包含的。否则返回数组的长度是1，元素是整个字符串
     	4.concat:链接两个字符串。
     	5.search:返回匹配的第一个的索引
     	6.match:括号里面放正则表达式，返回匹配到的字符串
     	7.slice:同数组一样
     	8.indexOf:与字符串一样
     	9.replace:替换，第一个参数是将要被替换的，第二个参数是新的
